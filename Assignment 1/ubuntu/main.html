Ubuntu Server


The server we are using in this course is Ubuntu. Specifically, it is:


Welcome to Ubuntu 14.04.1 LTS (GNU/Linux 3.13.0-45-generic x86_64)

As is obvious from the above welcome prompt when logging in, Ubuntu is a GNU/Linux based server. As such, it is free, and available directly from Ubuntu. The current version with a long view is 14.04.2. Note, however, that while many new servers are likely to benefit from running that version, it may not necessarily be advantageous for older installations to upgrade, as certain aspects of the server may not like the upgrade.

For more information, take a look at some of the breakdowns and specifications here.
