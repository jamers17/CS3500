Ubuntu Server Security References


Below is a list of useful references on maintaining a secure Ubuntu server.


  • Digital Ocean Initial Setup and Recommended Steps - Digital Ocean is a company that provides a popular cloud server service. Included in these initial steps is a breakdown of how to setup a basic firewall for security.

  • Ask Ubuntu - A popular discussion site for both Ubuntu users and developers, Ask Ubuntu has a user base regulated answer system to questions that generally produces appropriate responss. This particular answer is well vetted and includes tips such as setting up a brute force SSH blocker.

  • Essential Security for Linux Users - This is a very well written blog post on a particular set of steps that one administrator uses with every server. In addition to many of the suggestions from the links above, this blog also recommends public key authentication as a substitute for passwords for user accounts.
