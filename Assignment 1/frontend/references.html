Frontend References


Below is a list of useful references for HTML, CSS, & JavaScript


  • HTML & CSS Book - An HTML and CSS programming book by Jon Duckett. The book is very graphic heavy which helps to simplify complex concepts.

  • JavaScript & JQuery Book - another book by Jon Duckett in the same format but the subject of this book is JavaScript and JQuery

  • W3 Schools - An abundance of online tutorials focused on many web technologies, including HTML, CSS & JavaScript

  • Khan Academy - Interactive programming tutorials in all sorts of subjects. The link provided is a direct link to their Web Technologies tutorials. These tutorials cover HTML, CSS, & JavaScript

  • HTML Tags Reference - A quick reference guid for all HTML tags.

  • SitePoint - This website provides web programming news, tutorials, and a large community forum for discussion.

  • Official W3C Webpage - The standards organization for the World Wide Web

  • W3C Wikipedia Page

  • Official Standards For HTML Programming

  • Official Standards For CSS Programming

  • Official Standards For ECMAScript Language Programming
