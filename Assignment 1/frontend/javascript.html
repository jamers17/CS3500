JavaScript


JavaScript is a formalized scripting language in the Ecma Standard (Standard ECMA-262, Edition 5.1), which is based upon several scripting languages including JavaScript. ECMAScript itself was designed to be a web scripting language. It is multi-paradigm, supporting object-oriented programming styles as well as imperative and functional.


It is primarily used as a part of web browsers, which allow for client-side scripts to interact with the user, control the browser, communicate asynchronously and alter the document content that is displayed. In addition to these front end applications, it can also be used in server-side network programming with runtime environments, making it a useful tool for back end development as well.


Syntax & Examples


JavaScript allows developers to mainupulate the document content in ways that simple HTML and CSS eiher cannot do easily, or at all. For example, a simple drag and drop function in JavaScript is straightforward.


HTML code:

<img id="koala" ondragstart="drag(event)" height="200px" width="200px" src="../images/koala2.png">

<div id="koala2" ondrop="drop(event)" ondragover="allowDrop(event)"></div>

<div id="koala2" ondrop="drop(event)" ondragover="allowDrop(event)"></div>

JavaScript code:

function allowDrop(ev) {
     ev.preventDefault();
}

function drag(ev) {
     ev.dataTransfer.setData("text/html", ev.target.id);
}

function drop(ev) {
     ev.preventDefault();
     var data = ev.dataTransfer.getData("text/html");
     ev.target.appendChild(document.getElementById(data));
