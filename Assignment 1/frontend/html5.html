HTML5


The main language of the world wide web, the hypertext markup language, or HTML, is currenty in its 5th version. Its standards are maintained by W3 here.


Tools & Editors


There are a number of useful tools for HTML5. Perhaps the most useful is the official validator for HTML, which will detect any usages of the language which are not officially supported. Verifying the integrity of HTML code helps to facilitate greater compatibility of your front end design with different browsers and versions.


Additionally, there are a number of editors that work well for HTML. They range in features and complexity from simple text programs like notepad, simple programming editors like gedit, to higher end programming plaforms like Notepad++ and Sublime Text, to full blown IDEs (integrated development environment) like Netbeans.


Syntax & Examples


HTML uses tags, typically including a start and end tag that enclose content of a specific type dictated by the tag. This content could be a paragraph, a picture, a link, or even some JavaScript code. For example, here is the simple paragraph tag for text formatting:


<p>This is a paragraph in HTML.</p>

Notice the closing tag at the end. This tells the web browser that what follows is not a part of this paragraph. This is the result:


This is a paragraph in HTML.


...and the following is the HTML code for displaying an image:


<img src="../images/Koala.png" alt="a koala">

Notice that the entire line, including the URL of the image is part of that tag. The 'alt' or alternative text is provided for browsers that cannot load the image for any reason.
