Apache References


Below is a list of useful references for the Apache Web Server.


  • Apache Organization Homepage - Aphache organization official website. This website provides many references such as projects, guides, recommended books, and a blog.

  • Crappy Video - Although this video has poor production quality, mostly the sythasized speech, it still provides a lot of easy to understand facts about the Apache webserver platform.

  • Apache Wikipedia Page - a greate resource with lots of information about Apache

  • Apache Installation Guide - A comprehensive guide on installing Apache to a server, available in multiple languages.

  • Apache Core Features Reference Guide - A quick reference guid for Apache provided by the Apache organization.

  • Web Server Statistics - Apache powers more than half of the web servers in the world.
