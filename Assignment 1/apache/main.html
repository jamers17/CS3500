HTML5 Standards


HTML and its latest version, HTML5, are standardized through the efforts of W3.org. The current standard for HTML5 is here.

HTML Editors


HTML is a text based language, so there are a large number of editors that will work well for writing and editing HTML, from basic programs such as Notepad, TextEdit or gedit, depending on operating system, to professional grade editors like Sublime Text.

HTML Tools


Compatibility


When developing webpages using HTML, it is important that the developer check how their site looks and operates in different browsers. Due to the differences in browser implementation of web standards, one should view their site across a host of different browsers, platforms, and screen resolutions. Doing this manually would be very hard, which is why there are many online compatibility suites available for use, as well as the W3C official validator.


W3C Official Validator

Compatibility Suites


Many compatibility suites are available that will test a site and give results on any errors or formatting issues encountered. Some suites will even take screen shots of the results. For example, the suite Browsershots is easy to use. Using this on an earlier version of our own web page, we see screen shots of Chrome and Internet Explorer. Other compatibility suites include Browserstack and CrossBrowser Testing. These sites have free trials, but require a payment to access the full potential.
