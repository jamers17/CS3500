CS3500: Web Technologies


CS 3500 is a WMU class in which students learn how to setup and administrate a web server using Ubuntu, Apache and Ruby on Rails. This website serves as a reference point for topical information as it relates to the administration of a web server.


Of particular importance is the question of how to maintain a secure server in light of increasing malicious activity.


This project has been drawn extensively from a variety of referenced resources for design philosophy and code examples under the fair use license for educational purposes.
