Rails


Rails is a web application development framework written in the Ruby language. It is designed to make programming web applications easier and has two major guiding principles. First, don't repeat yourself by writing the same information over and over again. Second, convention over configuration. Rails is opinionated software and defaults to practical conventions for how to best accomplish tasks.
