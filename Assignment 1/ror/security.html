Ruby on Rails Security References


Below is a list of useful references on keeping Ruby on Rails secure.


  • Ruby on Rails Security Guide - A guide to keeping your Ruby on Rails applications secure written by the Ruby on Rails organization

  • Ruby on Rails Security Vulnerablities - CVE Details is a website that keeps a database of known security vulnerabilities. For each vulnerablity to provide the number of known exploits, dates, vulnerability score, access level gained, acces, complexity, and a few more details. Linked above is a direct link to the Ruby on Rails page.

  • Ruby on Rails Example Hack - This is an example of why it's important to make sure your application is written with the latest security standards, but also how it's just as important to update the server to protect your server from old vulnerabilities.
